# minesweeper
c语言扫雷游戏，大一期末答辩大作业
*************
## 基本功能简介
> 1. **点击方格打开方块**
> 2. **如果点开的方格是雷则失败**
> 3. **如果点开的方格不是雷，而周围有雷，则显示周围雷的数目**
> 4. **如果点开的方格不是雷，而方格周围没有雷，则扩展搜索雷**
> 5. **右键旗子标记雷**
> 6. **计时与暂停功能**
> 7. **保存最高记录**
> 8. **数字周围旗子数等于数字时自动打开周围格子**
> 9. **点击数字显示周围未开格子**
